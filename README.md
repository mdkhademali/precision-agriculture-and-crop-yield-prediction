- MATLAB Modeling

## Research Objectives

1. Predict crop yield using environmental and soil variables
2. Explore machine learning methods for agriculture
3. Build a reproducible research workflow

## Features

- Random Forest ML model
- Data preprocessing pipeline
- Model evaluation
- MATLAB implementation
- Notebook for experimentation

## Dataset Variables

- Soil moisture
- Temperature
- Humidity
- Rainfall
- Soil pH
- Nitrogen
- Phosphorus
- Potassium

## Future Extensions

Possible advanced research:

- Satellite NDVI integration
- Google Earth Engine crop analysis
- Deep learning crop disease detection
- Climate change crop modeling
- Smart irrigation prediction
